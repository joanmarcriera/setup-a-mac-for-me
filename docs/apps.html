Apps

The app list is small enough to remember and explicit enough not to drift.

Publishing the app roster matters because the repo doubles as documentation. The fastest way to keep it honest is to be explicit about both the tools that stay and the ones that are gone.

Keyboard Maestro handles the launcher and clipboard role. Rectangle Pro handles window management. Obsidian, Logseq, MailMate, Shottr, Kap, and Vivaldi stay in the everyday path.

Current App Groups

These match the generated Homebrew groupings in the repo.

Browsers + AI

Vivaldi, DuckDuckGo, VS Code, Codex, Claude, Antigravity.

Productivity

Keyboard Maestro, Rectangle Pro, Karabiner-Elements, Obsidian, Logseq, MailMate, KeePassXC, Spotify.

Backup

Kopia CLI plus KopiaUI for encrypted backups with Google Drive and NFS-friendly repository layouts.

Capture

Shottr, Kap, and ffmpeg.

Network

RustDesk, NetSpot, and the CLI diagnostics that only need to appear on heavier rebuilds.

Replacements And Watchlist

The public list stays useful only if stale tools are called out explicitly.

KeeWeb → KeePassXC

KeeWeb stayed valid in Homebrew but its desktop releases have stalled. KeePassXC is the chosen replacement.

MailMate

MailMate stays in the setup, but the stable Homebrew cask currently requires Rosetta 2 on Apple Silicon.

Kap

Kap stays. It still has a current Homebrew cask and remains a clean lightweight recorder.

Explicit Exclusions

Keeping the “no” list visible is part of keeping the rebuild stable.

  • App Expose is off.
  • Edge is out. Vivaldi is the browser path.
  • Alfred and Raycast are out. Keyboard Maestro covers that ground.
  • pnpm is out. Volta stays in.
  • Dropover, iBar, Whimsical, and Notion are out of the default rebuild.

Worth Evaluating Later

These are not part of the default rebuild, but they fit the way this machine is actually used.

Tailscale

Useful if you want painless access to NFS-backed machines, support boxes, or remote services without opening ports.

OrbStack

A strong fit if local Docker or Linux workloads matter and you want less friction than the standard Docker Desktop path.

BetterDisplay

Worth it if external monitor scaling, DDC brightness control, or awkward display layouts keep showing up.

Little Snitch

Good if you want tighter visibility into outbound traffic on a Mac that also does network and support work.
