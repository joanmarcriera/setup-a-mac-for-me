Restore State

Installation is fast. State restore is where the machine becomes yours again.

Homebrew gets the apps on disk. This page is the short manual list for the things that still have to be signed into, licensed, or nudged back into the right shape.

Keep this page close while rebuilding. It is the part most likely to be forgotten after a few months away.

Core Restore Checklist

Do these in roughly this order so the machine starts feeling normal quickly.

  • Sign into Vivaldi and confirm sync brings back workspaces, bookmarks, and extensions.
  • Sign into VS Code and enable Settings Sync.
  • Reconnect Codex and Claude to the repos and workflows you use daily.
  • Import or sync Keyboard Maestro macros and reconnect launcher and clipboard shortcuts.
  • Activate Rectangle Pro and restore the expected window movement shortcuts.
  • Reconnect Karabiner-Elements rules if the keyboard layer is part of the current machine.
  • Open Obsidian and reconnect vaults and plugins.
  • Check Shottr and Kap preferences if output folders or shortcut bindings matter.

Things Worth Storing Outside This Repo

Document the restore source, not the secret itself.

Licenses

Keyboard Maestro and Rectangle Pro license material should live in a secure password manager, not in git.

Exports

Macro exports, browser backup flows, and Karabiner profiles should live in a separate private backup path.

Account-heavy setups

Keep a note of which logins must be restored, but avoid turning the repo into a secrets checklist.
