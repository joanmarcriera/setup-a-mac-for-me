Homebrew

Pick a preset, trim the groups, copy the command.

The installer is backed by the same JSON file that generates the repo Brewfiles. If the selection matches a preset, you get one clean brew bundle command. If not, the UI falls back to grouped slices or merged raw install commands.

The default path is Workstation. It includes the daily tools but leaves the heavier network kit out until it is actually needed.
